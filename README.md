# DeepSpeed Windows 预编译包 (WHL)

本项目提供在 Windows 上编译好的 DeepSpeed WHL 文件，让你无需复杂环境即可直接安装使用。

## 🔍 项目说明

- **目标**：为 Windows 用户提供开箱即用的 DeepSpeed 预编译包
- **适用**：Python 3.12
- **非官方**：本项目仅提供预编译包，DeepSpeed 本身由 Microsoft 维护
